# Changelog

All notable changes to the Freedom Hugo theme will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - Initial Development

### Features
- Simple and card layout styles for post listings.
- Menu widget with active state support.
- Responsive CSS with custom properties and theming.
- Dark/Light mode support based on user preference.
- Extra Markdown features supported using Goldmark Extras.
- Shortcodes for icon, expand, spacing, alignment, icons, full & wide content.

### Technical Details
- Hugo v0.146.0+ required
- Configuration: `hugo.yaml` format
